# Icons Klasörü

Bu klasör Chrome eklentisi için icon dosyalarını içerir.

## Icon'ları Oluşturma

Ana dizindeki `generate-icons.html` dosyasını tarayıcınızda açın ve icon'ları otomatik oluşturup bu klasöre kaydedin.

## Gerekli Dosyalar

- `icon16.png` - 16x16 piksel (toolbar icon)
- `icon48.png` - 48x48 piksel (eklenti yönetimi)
- `icon128.png` - 128x128 piksel (Chrome Web Store)

## Alternatif

Kendi icon'larınızı da kullanabilirsiniz. Sadece yukarıdaki boyutlarda PNG formatında olduklarından emin olun.
